[OpenAI Codex](https://github.com/openai/codex) is OpenAI's CLI agent for software development. You can connect it to TryPost's MCP server to manage your social media scheduling with natural language.

## Prerequisites

- Codex CLI installed (`npm install -g @openai/codex`)
- A TryPost API key

## Setup

<Steps>
  <Step title="Set your API key as an environment variable">
    Add your TryPost API key to your shell profile (`~/.zshrc`, `~/.bashrc`, etc.):

    ```bash
    export TRYPOST_API_KEY="YOUR_API_KEY"
    ```

    <Note>
      Replace `YOUR_API_KEY` with your actual API key from the [TryPost dashboard](https://app.trypost.it).
    </Note>
  </Step>
  <Step title="Open the configuration file">
    Edit `~/.codex/config.toml` and add the TryPost server:

    ```toml
    [mcp_servers.trypost]
    url = "https://app.trypost.it/mcp/trypost"
    bearer_token_env_var = "TRYPOST_API_KEY"
    ```
  </Step>
  <Step title="Restart Codex">
    Close and reopen Codex to load the new configuration.
  </Step>
  <Step title="Verify the connection">
    Ask Codex to interact with your TryPost account:

    ```
    List all my TryPost posts
    ```
  </Step>
</Steps>

<Tip>
  Codex uses TOML for configuration (not JSON). It provides `bearer_token_env_var` to securely reference API keys from environment variables instead of hardcoding them in the config file.
</Tip>

Create a new draft post for all my social accounts
