# Supported Platforms

TryPost supports 12 social networks — 14 connection types in total, since LinkedIn and Instagram each have two flavors:

| Platform | Connection types | Content types |
|----------|------------------|---------------|
| [LinkedIn](/platforms/linkedin) | Profile (`linkedin`) and Company Page (`linkedin-page`) | Posts, carousels |
| [X](/platforms/x-twitter) | `x` | Posts with images or video |
| [Facebook](/platforms/facebook) | `facebook` | Posts, reels, stories |
| [Instagram](/platforms/instagram) | Standalone (`instagram`) and via Facebook Business (`instagram-facebook`) | Feed, carousels, reels, stories |
| [TikTok](/platforms/tiktok) | `tiktok` | Video and photo carousels |
| [YouTube](/platforms/youtube) | `youtube` | YouTube Shorts |
| [Threads](/platforms/threads) | `threads` | Posts with images or video |
| [Pinterest](/platforms/pinterest) | `pinterest` | Pins, video pins, carousels |
| [Bluesky](/platforms/bluesky) | `bluesky` | Posts with media |
| [Mastodon](/platforms/mastodon) | `mastodon` | Posts with media (any instance) |
| [Telegram](/platforms/telegram) | `telegram` | Channel & group posts with media albums |
| [Discord](/platforms/discord) | `discord` | Server channel messages with media, mentions & embeds |

## Connecting accounts

On **TryPost Cloud**, simply go to **Accounts** in the dashboard and click the platform you want to connect. You'll be redirected to authorize TryPost.

<Note>
  **Self-hosting?** You'll need to create developer apps and configure API credentials for each platform. See the [Self-Hosting configuration guide](/self-hosting/configuration) for details.
</Note>
