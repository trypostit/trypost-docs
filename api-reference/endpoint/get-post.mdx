---
title: "Get post"
api: "GET /posts/{post}"
description: "Get a specific post by ID with all its platform entries, media, and labels."
---

## Request

<ParamField path="post" type="string" required>
  The post UUID.
</ParamField>

## Response

Returns the full post. Response shape is identical to one entry in [`GET /posts`](/api-reference/endpoint/list-posts) — see that endpoint for full field documentation.

<RequestExample>
```bash
curl -X GET https://app.trypost.it/api/posts/9f1a2b3c-4d5e-6f7a-8b9c-0d1e2f3a4b5c \
  -H "Authorization: Bearer YOUR_API_KEY"
```
</RequestExample>
