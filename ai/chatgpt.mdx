[ChatGPT](https://chatgpt.com) can connect to TryPost's MCP server as a developer-mode app, so you can manage social media scheduling with natural language in chat.

## Prerequisites

- A ChatGPT account with [Developer mode](https://developers.openai.com/api/docs/guides/developer-mode) available (availability depends on your plan and workspace policy)
- A TryPost Cloud account with an active trial or subscription ([start a 7-day free trial](/knowledge-base/plans-and-billing))

## Setup

<Steps>
  <Step title="Enable Developer mode">
    In ChatGPT, open **Settings** → **Security and login** and turn on **Developer mode**.

    On Business / Enterprise / Edu workspaces, an admin may need to allow Developer mode first (for example under workspace **Permissions & Roles** or **Apps → Advanced settings**). If the toggle is missing, ask your workspace admin.
  </Step>
  <Step title="Create a developer-mode app">
    Open **Settings → Plugins** (or go to [chatgpt.com/plugins](https://chatgpt.com/plugins)) and click the **+** button to create a developer-mode app.

    Fill in:

    | Field | Value |
    |-------|-------|
    | **Name** | `TryPost` |
    | **Description** | `Schedule and publish social posts with TryPost` |
    | **MCP server URL** | `https://app.trypost.it/mcp/trypost` |

    Leave client ID / secret blank if ChatGPT offers them — TryPost supports Dynamic Client Registration and discovers OAuth from the URL.
  </Step>
  <Step title="Sign in and scan tools">
    Create the connection. If ChatGPT prompts for authorization, sign in to TryPost and click **Allow**. Wait for the tool scan to finish so ChatGPT discovers the TryPost tools.
  </Step>
  <Step title="Use TryPost in a conversation">
    Start a new chat, open the **+** / tools menu, choose **Developer mode**, and enable the **TryPost** app for that conversation. Then ask:

    ```
    List all my scheduled posts
    ```

    ChatGPT will call the TryPost tools to fetch your posts.
  </Step>
</Steps>

<Note>
  ChatGPT only connects to remote HTTPS MCP servers from OpenAI's infrastructure. The hosted endpoint `https://app.trypost.it/mcp/trypost` works out of the box. Self-hosted instances must be publicly reachable over HTTPS (or use a tunnel) — local `stdio` MCP servers are not supported.
</Note>

<Tip>
  After you change anything on the MCP server side, open the app under **Settings → Plugins**, click **Refresh**, and start a new conversation so ChatGPT picks up updated tools and descriptions.
</Tip>
